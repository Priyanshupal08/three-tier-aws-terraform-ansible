# 🚀 Three-Tier AWS Architecture using Terraform & Ansible

## 📖 Overview

This project demonstrates the deployment of a secure and scalable Three-Tier Architecture on AWS using Terraform for Infrastructure as Code (IaC) and Ansible for configuration management.
